# Python Bugfix Demo

A small Python debugging demo that shows how a broken script can be diagnosed and fixed.

![Python](https://img.shields.io/badge/Python-3.11-blue)
![Debugging](https://img.shields.io/badge/Use%20Case-Bugfixing-red)
![CLI](https://img.shields.io/badge/Interface-Command%20Line-orange)

## What this project does

This repository demonstrates a simple Python bugfix workflow:

- a broken script with common coding mistakes
- a corrected version of the same script
- clear before/after comparison

## Files

- `broken_app.py` — contains intentional bugs
- `fixed_app.py` — corrected and working version
- `screenshots/bugfix-demo.png` — visual preview

## Example Bugs Fixed

- incorrect variable name
- undefined variable usage
- runtime error prevention
- logic cleanup

## Usage

Run the broken script:

```bash
python broken_app.py
